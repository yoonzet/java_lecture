## 단축키
|단축키| 기능 |
| --- | --- |
| ⌘ + shift + L |  단축키 전체 목록보기 |
| ⌘ + D |  한 줄 삭제 |
| ⌘ + ⌥ + down |  행단위 복사 |
| ⌥ + shift + A  | 멀티컬럼 편집 |
| ⌥ + up, down | 행단위 이동 |
| ⌘ + i | 자동 들여쓰기 |
| ⌥ + spacebar (ctrl+spacebar였으나 바꿈 *하단참조) | 탬플릿 자동완성 (예: sysout) |


## 단축키 바꾸기
